VIP 服务器¶

Time:2 分钟

VIP 服务器是私人游戏服务器 。如果游戏允许使用 VIP 服务器，则玩家可以对其进行购买，并按照此文决定可以与之共同游戏的其他玩家。

设置 VIP 服务器¶

VIP 服务器可为游戏中的特定/articles/games and places|场景设立。发布游戏后，将会默认开启 VIP 服务器选项。需要对 VIP 服务器进行设置时，请遵循以下步骤：

  1. 进入场景的配置页面。

  2. 单击 Access 选项卡。

  3. 在页面底端可以看到 VIP 服务器的启用与禁用选项，并对其设置价格。

  4. 单击 Save 按钮进行保存。

如果你的游戏包含多个场景，且希望为整个游戏开启 VIP 服务器选项时，我们建议开发者为游戏的起始场景设置 VIP 服务器。

注意：更改 VIP 服务器价格后，所有正在进行的订阅都将被取消。拥有订阅的玩家将会在其收件箱中收到价格变更的讯息。
