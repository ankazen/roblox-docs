条件结构

10 分钟

使用条件结构后，脚本将可以在满足特定条件时进行特定操作。若希望对这些条件进行进一步了解，请查看下表中总结概述的关系运算符：

运算符 描述 示例
== 等于 3 == 5  →  false
~= 不等于 3 ~= 5  →  true
> 大于 3 > 5  →  false
< 小于 3 < 5  →  true
>= 大于或等于 3 >= 5  →  false
<= 小于或等于 3 <= 5  →  true

条件测试类结构

if — then

基础的 if—then 语句可以在特定条件为 true 时运行指定代码块。请参考下列示例：

Relational Operator Example 预期输出 展开 
if 2 + 3 == 5 then
	print("two plus three is five")
end
two plus three is five
复制代码 亮色模式

需要注意的是，Lua 语言会将“非 true 情况”等同于 false 情况。也就是说，其会将 false 和 nil 都当做非 true 情况处理：

代码示例 预期输出 展开 
local nilVariable  -- Variable is declared but has no value, so it's nil
local falseVariable = false  -- Variable is declared with value of false

if nilVariable ~= true then
	print(nilVariable)  -- Outputs "nil" since nil is not true
end

if falseVariable ~= true then
	print(falseVariable)  -- Outputs "false" since false is not true
end
nil
false
复制代码 亮色模式

elseif — then

当将 elseif—then 语句添加至 if—then 结构时，代码将会在前一个条件为 false 的情况下继续检查其他条件是否为 true。Lua 语言将会从上往下进行检查，直到发现第一个为 true 的条件时为止，并运行该条件的指定代码块。

Elseif Example 预期输出 展开 
if 10 > 100 then 
    print("10 is greater than 100") 
elseif 10 > 50 then	
    print("10 is greater than 50") 
elseif 10 < 100 then	
    print("10 is less than 100") 
end
10 is less than 100
复制代码 亮色模式

else

当使用 else 结束条件结构时，在前置所有条件都为被判定为 true 时将会执行 else 所指定的代码块。以下列代码为例，10 > 100 和 10 > 25 两个条件都为 false，因此结果运行的是 else 的代码块，而不是前两个条件的代码块。

Else Statement Example 预期输出 展开 
if 10 > 100 then 
    print("10 is greater than 100") 
else 
    print("10 is less than 100") 
end
10 is less than 100

条件循环类结构

使用条件循环后，当特定条件为 true 时将可以执行指定代码；同时也可以重复执行代码，直到特定条件为 true 为止。

while — do

while—do 循环将会判定特定条件为 true 或 false。当为 false 时，将会结束循环并继续执行循环后的代码。当为 true 时，将会执行 do 与 end 之间的代码块，并在执行完毕之后再次对条件的 true 或 false 状态进行判定。

代码示例 预期输出 展开 
local timeRemaining = 10

while timeRemaining > 0 do
	print("剩余秒数： " .. timeRemaining)
	wait(1)
	timeRemaining = timeRemaining - 1	
end

print("倒数计时已归零！")

repeat — until

repeat—until 循环将会持续重复执行，直到满足特定条件为止。注意：由于条件测试会在 repeat 和 until 之间的代码运行后进行，此代码将会至少执行 1 次。

代码示例 预期输出 展开 
local currentGoblinCount = 18

-- 在游戏中最多生成 25 个哥布林
repeat
	spawnGoblin()
	currentGoblinCount = currentGoblinCount + 1
	print("当前哥布林数目： " .. currentGoblinCount)
until currentGoblinCount == 25

print("哥布林生成完毕！")

逻辑运算符的使用

多条件测试类结构

为了避免多个条件测试重复进行，开发者可以通过逻辑运算符 and 和 or 进行多条件测试。举例来说，下列结构将会测试两个条件是否为 true：

代码示例 预期输出 展开 
local pasta = true
local tomatoSauce = true

if pasta == true and tomatoSauce == true then
	print("We have spaghetti dinner!")
else
	print("Something is missing...")
end
We have spaghetti dinner!
复制代码 亮色模式

开发者同时也可以通过组合逻辑运算符进行更为复杂的逻辑测试。举例来说，下列代码将会检查前两个条件是否为 true 或 第三个条件是否为 true：

代码示例 预期输出 展开 
local pasta = false
local tomatoSauce = true
local garlicBread = true

if (pasta == true and tomatoSauce == true) or garlicBread == true then
	print("We have either spaghetti dinner OR garlic bread!")
else
	print("Something is missing...")
end
We have either spaghetti dinner OR garlic bread!

Non-Truth Tests

While the relational operators can be “flipped” to test for non-truth conditions, the logical operator not is useful for testing the “absence of true” (either false or nil).

代码示例 预期输出 展开 
local nilVariable  -- Variable is declared but has no value, so it's nil
local falseVariable = false  -- Variable is declared with value of false

if not nilVariable then
	print(nilVariable)  -- Outputs "nil" since nil is not true
end

if not falseVariable then
	print(falseVariable)  -- Outputs "false" since false is not true
end
nil
false
复制代码 亮色模式

The not operator can also test for the opposite of an entire multi-condition statement. For example, the following code confirms that there are not more than 25 goblins nor is the player’s experience level less than 5.

代码示例 预期输出 展开 
local currentGoblinCount = 18
local playerExperienceLevel = 6

if not (currentGoblinCount > 25 or playerLevel < 5) then
	print("Spawn more goblins!")
end
Spawn more goblins!
