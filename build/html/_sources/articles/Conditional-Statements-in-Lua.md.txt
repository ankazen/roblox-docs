# 条件结构 
Time:<em>10  分钟</em>

使用条件结构后，脚本将可以在满足特定条件时进行特定操作。若希望对这些条件进行进一步了解，请查看下表中总结概述的关系运算符：

运算符 描述 示例

`==`
等于
`3 == 5` → **false**

`~=`
不等于
`3 ~= 5` → **true**

`>`
大于
`3 > 5` → **false**

`<`
小于
`3 < 5` → **true**

`>=`
大于或等于
`3 >= 5` → **false**

`<=`
小于或等于
`3 <= 5` → **true**
